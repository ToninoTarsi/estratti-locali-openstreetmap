==INFORMATION ON SHAPE FILES==

Files contained here are coded following
the WGS_1984_UTM_Zone_32N coordinate system.
then EPSG:32632
http://spatialreference.org/ref/epsg/wgs-84-utm-zone-32n/

You can find the following layers:

 * addresses
 * roads
 * waterways
 * water
 * glaciers
